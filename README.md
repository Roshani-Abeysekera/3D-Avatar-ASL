# 3D Avatar ASL Translator

Convert speech or text into American Sign Language (ASL) movements in real-time to improve accessibility.

## Demo

[![Watch Demo](https://img.youtube.com/vi/GFXoy6Giw10/0.jpg)](https://youtu.be/GFXoy6Giw10)

Click the image above to watch the demo video.

## Project Overview

- Full-stack application converting speech/text into ASL movements.
- Real-time frontend interface for user input and sign visualization.
- Backend logic using Python and NLP techniques for text processing.
- Interactive and testable UI designed for accurate and predictable output.

## Technologies Used

- **Frontend:** JavaScript, HTML, CSS  
- **Backend:** Python, NLP libraries
